2D Forager like Renderer
Uses Roblox image frames -> their positions are modified each frame.
The grass tile connections simply use a table that contains possible
connection directions and adjusts the texture accordingly, this gives it
a unified look.

Minecraft-like Terrain Generator
Using some known 3D perlin noise generation techniques and some
fractal brownian motion, I was able to get this kind of terrain
generation. I also included a culling system which really helps
when you're using Roblox decals and parts to render the blocks
